# FanClash (팬클래시) — 스트리머 인터랙티브 도구

## 개요

치지직, 숲, 틱톡 등에서 방송하는 스트리머를 위한 시청자 참여 & 경쟁 유도 인터랙티브 도구.
팬들 간의 경쟁심을 자극해 후원을 증가시키는 것이 핵심 가치.

### 타겟
- 1차: 여캠 BJ (경쟁/남심 자극 기능 특화)
- 2차: 전체 스트리머 (범용 인터랙티브 기능)

### 제품 형태
- OBS 오버레이 위젯 + 스트리머 대시보드 (Phase 1)
- 시청자 전용 웹앱 (Phase 2)
- 채팅 명령어 하이브리드 (Phase 2)

---

## 아키텍처

```
[스트리머 대시보드]  ←→  [실시간 서버 (WebSocket)]  ←→  [OBS 오버레이 위젯]
       웹앱                    Socket.io                  브라우저 소스
                                  ↕
                          [시청자 웹앱] (Phase 2)
```

### 기술 스택
- **프론트엔드**: Next.js (대시보드 + 오버레이 페이지)
- **실시간**: Socket.io (순위 변동, 알림 즉시 반영)
- **DB**: Supabase (PostgreSQL + 인증 + 실시간 구독)
- **배포**: Vercel (웹) + Railway 또는 Fly.io (WebSocket 서버)

### 작동 흐름
1. 스트리머가 대시보드에서 회원가입 → 위젯 생성/설정
2. OBS 브라우저 소스 URL 복사해서 방송에 추가
3. 방송 중 대시보드에서 후원 정보 입력 (또는 API 자동 연동)
4. 위젯이 실시간으로 랭킹/이펙트/알림 반영

## MVP 기능 상세 (Phase 1)

### 위젯 1: 실시간 후원 랭킹 보드
- TOP 5 표시, 순위 변동 시 애니메이션 (올라가면 금빛, 내려가면 흔들림)
- 1등에 왕관 아이콘 자동 부여
- 오늘/이번 주/이번 달 기간 전환 가능
- **비주얼**: 순위 카드, 1등 왕관 + 금빛 테두리, 슬라이드 애니메이션

### 위젯 2: 왕좌 쟁탈전
- 1등이 바뀌는 순간 화면 전체 이펙트 + 알림
- "XXX님이 왕좌를 빼앗았습니다!" 연출
- 쟁탈 횟수 카운터 (경쟁 분위기 강화)
- **비주얼**: 풀스크린 알림, 왕관 넘어가는 연출

### 위젯 3: 도네이션 목표 게이지
- 단계별 목표 설정 (예: 5만→10만→30만→50만)
- 각 단계 달성 시 스트리머가 설정한 미션 텍스트 표시
- 게이지 애니메이션 + 달성 시 축하 이펙트
- **비주얼**: 프로그레스바 + 마일스톤 아이콘, 달성 시 폭죽

### 위젯 4: 호감도/친밀도 시스템
- 팬별 누적 후원에 따라 호감도 레벨 상승
- 레벨별 칭호: 지나가는 팬 → 단골 → 열혈팬 → 첫사랑 → 소울메이트
- 칭호는 스트리머가 커스텀 가능
- 레벨업 시 팝업 알림
- **비주얼**: 채팅 옆 뱃지 + 레벨업 팝업

### 위젯 5: 후원 배틀
- **스트리머가 배틀 개설**: 대시보드에서 베네핏 설정 ("이기면 듀오 한판!") + 참가 최소 도네 금액 설정
- **모집 화면 표시**: 오버레이에 "배틀 모집 중! 도네로 참가하세요" + 베네핏 표시
- **시청자가 도네로 참가 신청**: 참가자 입장 시 이펙트
- **배틀 시작**: 제한 시간 내 후원 대결, 실시간 대결 화면
- **승자 확정**: 이펙트 + 베네핏 안내 표시
- **승자 베네핏 (스트리머 커스텀 설정)**: 듀오/게임, 노래 신청권, 질문 답변권, "챔피언" 특별 칭호, 명예의 전당 등재 등
- **수익 포인트 3중**: 참가 도네 → 배틀 중 후원 경쟁 → 관전 시청자 응원 도네
- **비주얼**: VS 화면 분할, 실시간 금액 바, 카운트다운 타이머

### 위젯 6: 팀 대결
- 시청자를 2~4팀으로 자동/수동 배정
- 팀별 후원 총액 실시간 비교 바 그래프
- 승리팀 전원 칭호/뱃지 부여
- **비주얼**: 팀 컬러 바 그래프, 역전 시 흔들림 이펙트

### 스트리머 대시보드
- 회원가입/로그인 (Supabase Auth)
- 위젯별 ON/OFF 및 설정 (색상, 테마, 칭호명 등)
- OBS 브라우저 소스 URL 복사 버튼
- 수동 후원 입력 (닉네임 + 금액)
- 배틀 개설/관리 (베네핏 설정, 최소 금액, 시간 제한)
- 실시간 현황 모니터링 (총 후원, 참여자 수 등)

## 테마 & 비주얼

### 테마 3종
1. **깔끔 모던 (기본)** — 화이트/다크 모드, 미니멀 카드 UI, 어떤 방송에나 어울림
2. **게임/배틀** — 어두운 배경 + 네온 컬러, 격투게임 HP바 스타일
3. **여캠 감성** — 파스텔톤, 하트/별/반짝이 이펙트, 부드러운 애니메이션

### 커스터마이징
- 위젯 색상, 폰트, 위치, 투명도 조절
- 테마팩 일괄 적용 또는 위젯별 개별 설정

## 수익 모델

| 모델 | 내용 |
|------|------|
| **프리 (무료)** | 위젯 2개까지, 기본 테마만, 로고 워터마크 |
| **프로 (월 9,900원)** | 위젯 무제한, 모든 테마팩, 워터마크 제거, 커스터마이징 풀 |
| **추후** | 수수료 모델 검토 가능 (배틀 도네이션의 X% 등) |

## Phase 계획

### Phase 1 (MVP — 현재)
- 스트리머 대시보드 (가입, 위젯 설정, 수동 후원 입력)
- OBS 오버레이 위젯 6종: 후원 랭킹, 왕좌 쟁탈전, 도네 목표 게이지, 호감도/칭호, 후원 배틀, 팀 대결
- 테마 3종 (모던/게임/여캠)

### Phase 2 (반응 보고 확장)
- 시청자 전용 웹앱 (폰으로 접속해서 참여)
- 투표/설문, 룰렛/뽑기, 퀴즈
- 치지직/숲 API 연동 (채팅 명령어 + 후원 자동 인식)

### Phase 3 (스케일업)
- 예측 베팅, 미니게임, 이모지 폭탄
- 틱톡 라이브 연동
- 통계/분석 대시보드
- 크리에이터 간 랭킹/커뮤니티
