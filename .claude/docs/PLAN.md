# MCN 프로젝트 — 크리에이터 프로덕트 시리즈

## 개요
치지직, 숲, 틱톡 등 스트리머/인플루언서를 위한 독립 프로덕트 여러 개를 만들어 시장 반응을 검증하는 프로젝트.

## 프로덕트 로드맵 (우선순위순)
1. **FanClash (팬클래시)** — 도네이션 이벤트 엔터테인먼트 레이어
2. **ClipForge (클립포지)** — 방송 하이라이트 자동 추출 → 틱톡/쇼츠 변환 ← MVP 완료
3. **ShieldChat (쉴드챗)** — 악플 수집 → 독성 분석 → 증거 보존 → 법적 대응 ← MVP 완료

## FanClash 상세

### 핵심 컨셉
FanClash는 **직접 결제/도네이션 플랫폼이 아니다.**
- 투네이션, 스트림랩스, 틱톡 등 **기존 도네이션 플랫폼의 API/라이브러리**를 통해 도네이션 이벤트를 감지
- 감지된 이벤트를 **배틀, 랭킹, 룰렛, 왕좌 등 재밌는 인터랙션으로 변환**하는 엔터테인먼트 레이어
- 돈의 흐름은 기존 플랫폼이 처리, FanClash는 **시각적·게임적 재미**만 담당

### 연동 플랫폼 (이벤트 감지 소스)
1. 투네이션 (alertbox_key)
2. 스트림랩스 (socket token)
3. 틱톡 라이브 (username)
4. 치지직 (DJ_ID)
5. 숲/아프리카TV (BJ_ID)

### Phase 구분
- **Phase 1 (현재):** 외부 도네이션 이벤트 감지 → 위젯 인터랙션 + Toss Payments 구독 결제 연동
- **Phase 2 (미래):** 직접 도네이션 기능 (자체 결제 흐름)
- **Phase 3 (미래):** 커스텀 테마 마켓 등 추가 수익화

### 사용자 구분
- **스트리머 (핵심 사용자):** FanClash에 가입, 플랫폼 연동, 위젯 설정, 대시보드 사용
- **팬:** FanClash에 직접 접속하지 않음. 기존 플랫폼에서 도네이션 → OBS 오버레이로 결과 확인
  - 스트리머가 랭킹 등을 공유 링크로 보내줄 때만 접속 가능 (Phase 2+에서 확장 검토)

### 수익모델 (Phase 1)
- Toss Payments를 통한 **구독 상품 결제** (프리미엄 플랜)
- 무료: 기본 위젯, 제한된 기능
- 유료 구독: 고급 위젯, 커스텀 테마, 상세 분석 등

## ClipForge 상세

### 핵심 컨셉
ClipForge는 **VOD/스트림 URL을 입력하면 AI가 자동으로 하이라이트를 감지하고 9:16 세로 클립을 생성**하는 도구.
- 지원 플랫폼: YouTube, 치지직(CHZZK), Twitch, 아프리카TV
- MVP는 FFmpeg WASM + Web Audio API로 클라이언트 사이드 실제 영상 처리
- Next.js 16 + TypeScript + Tailwind + Supabase

### 수익모델
- Free: 월 3개 클립, 워터마크 포함, 기본 자막
- Pro (9,900원/월): 무제한 클립, 워터마크 제거, 커스텀 자막, 우선 처리

### 주요 페이지
- 랜딩: 히어로 + 3단계 설명 + 기능 카드 + 요금제 미리보기
- 인증: 로그인/회원가입 (Supabase Auth)
- 대시보드: 사용량 통계, 최근 작업 목록
- 새 클립: 영상 파일 업로드 → Web Audio 하이라이트 감지 → FFmpeg WASM 클립 추출
- 자막 스타일: 5종 프리셋 (틱톡/쇼츠/인스타/게이밍/미니멀)
- 클립 관리: 그리드 뷰, 정렬, 다운로드/삭제
- 설정: 프로필, 플랜, 사용량
- 요금제: Free vs Pro 비교 + FAQ

## ShieldChat 상세

### 핵심 컨셉
ShieldChat은 **크리에이터를 위한 악성 댓글 방어 도구**.
- 악플을 수동 입력 또는 일괄 붙여넣기로 수집
- 한국어 키워드 기반 독성 분석 (욕설/협박/명예훼손/성희롱/차별 5개 카테고리)
- 심각도 4단계 (low/medium/high/critical) 자동 분류
- 증거 보존용 PDF 리포트 생성
- 법적 대응 가이드 (사이버수사대 신고, 고소장 작성, 증거 보존 방법)

### 기술 스택
- Next.js 16 + TypeScript + Tailwind + Supabase
- 한국어 독성 감지: 키워드 매칭 기반 (src/lib/toxicity.ts)
- PDF: html2canvas + jspdf 실제 PDF 다운로드

### 수익모델
- Free: 최대 100건 댓글, 수동 입력, 기본 리포트
- Pro (14,900원/월): 무제한, 일괄 입력, PDF 리포트, 자동 수집(Phase 2), 법률 상담 연결(Phase 2)

### 주요 페이지
- 랜딩: 히어로 + 3단계 + 기능 카드 + CTA
- 인증: 로그인/회원가입 (Supabase Auth)
- 대시보드: 통계 + 심각도 분포 차트 + 최근 위험 댓글
- 댓글 추가: 텍스트 입력 / 일괄 입력 모드
- 댓글 목록: 필터(심각도/카테고리/플랫폼) + 정렬 + 벌크 선택
- 댓글 상세: 분석 결과 + 증거 보존 + 메모
- 리포트: 목록 + 새 리포트(댓글 선택) + PDF 인쇄 페이지
- 법적 가이드: 5개 섹션 (신고/법률/고소장/증거/링크)
- 설정: 프로필 + 플랜 + 이용 현황
- 요금제: Free vs Pro
